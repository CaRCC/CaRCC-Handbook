## Development

To develop locally run the following to setup the software.
```bash
./setup.sh
```

And then run a local server available at http://localhost:8000
```bash
. ./.venv/bin/activate
mkdocs serve
```

To deploy to GitHub run
```bash
. ./.venv/bin/activate
mkdocs gh-deploy
```
